# Fractals

Draw beatiful fractals in your terminal with drawille

Package includes:

- LindenMayer System
- Examples

Dependancies
```
$ pip install drawille
```

# Examples

```
$ git clone https://github.com/A1rPun/Fractals
$ cd Fractals
$ python -m fractals.dragoncurve
```
